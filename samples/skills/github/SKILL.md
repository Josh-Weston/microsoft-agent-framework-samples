# GitHub

## When to use this skill

Use this skill when a user requests information about code, repositories, pull requests, or issues, you are interacting with GitHub.

## How to get information about GitHub

Do not attempt to answer the question from your own knowledge. You must immediately call the `github_skill` tool and pass the user's exact request to it.
